Installation and Running
------------------------

For running Antelope you must install the Java Runtime Environment version 6
or newer on your computer. Get the latest Java version from:

    http://java.com

You can always download the Antelope's latest version from:

    http://turing.iimas.unam.mx:8080/AntelopeWEB

Antelope does not need any special installation procedure, for using it simply
unpack the contents of the downloaded file on your hard disk, and:

    - on Windows and Mac OS, double-click the file EmbeddedAntelope.jar
    - on Linux and other Unices, execute the script run_antelope.sh
